Evitar Modificações:
Object.preventExtensions -> não permite que insira novos atributos em um objeto, somente permite alterações e delet.

Object.seal -> não permite inseriir e nem deletar, somente alterações.

Array Js

vetor.sort() -> organiza os elementos por ordem alfabética
vetor.splice(1,1,'nome') -> vetor(indice, retirar, colocar)
vetor.slice() -> cria um novo array a partir do elemento definido

forEach() -> Percorre e array. É passado sempre 3 parâmetros. 1º: nome do elemento, 2º: índice, 3º: array. Necessariamente nessa ordem.
É possível instanciar um forEach através do Array.prototype.function

map() -> Cria um novo array, com base em um array. 1º: valor, 2º: índice, 3º array.

filter() -> filtra dados de array.
